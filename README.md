## autogeometrypro

该节点作用为自动校正图像的透视，用于空间、建筑摄影行业。
该节点通过对图片的垂直线进行分析，选取合适的作为辅助线进行透视调整
最大修改角度为45度
矫正后不会减少原图内容，因此会多出一部分并且填充为白色
矫正后的图配合本人的内容识别填充工作流使用最佳
